# 데이터 전처리

- 데이터 분석 과정에서 데이터 전처리는 반드시 거쳐야 하는 과정
- 전처리 결과가 분석 결과에 직접적인 영향을 주고 있어서 반복적으로 수행함
- 데이터 분석의 단계 중 가장 많은 시간이 소요됨
- 데이터 정제 → 결축값 처리 → 이상값 처리 → 분석변수처리 순서로 진행
